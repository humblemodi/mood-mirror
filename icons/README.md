# Icons Directory

This folder contains the icons for the Mood Mirror Chrome extension.

## Required Icons

- `icon16.png` - 16x16px (toolbar icon)
- `icon48.png` - 48x48px (extension management page)
- `icon128.png` - 128x128px (Chrome Web Store)

## Creating Icons

You can create these icons using:
- Image editing software (Photoshop, GIMP, Figma)
- Online icon generators
- Export emojis at different resolutions

For local testing, place PNG files with the above names in this directory.
