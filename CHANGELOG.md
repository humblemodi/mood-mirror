# Changelog

All notable changes to Mood Mirror will be documented in this file.

## [1.1.0] - 2025

### Added
- Auto-Refresh Feature: Mood detection automatically refreshes at intervals (30min, 1hr, 3hr, 5hr, 12hr, 24hr)
- Browser Notifications: Alerts when mood drops below thresholds
- Advanced Actionables System: 6-category mood-based actionables with empathetic prompts
- Settings Panel: Customize auto-refresh intervals
- Global Helplines: India (TeleMANAS) and USA (988) crisis support
- Critical Prompt Enhancement: Encouraging message for critical mood states

## [1.0.0] - 2025

### Initial Release
- Real-time Mood Analysis using 10+ parameters
- 100% Privacy-First: Client-side only processing
- Minimalist UI design
- Daily Reports and trend tracking
- Multi-browser support (Chrome, Brave, Edge)
- Mental health crisis resources
